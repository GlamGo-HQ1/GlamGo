# GlamGo

**A premium visual hair marketplace with built-in payment protection — powered by Interswitch.**

---

## The Problem

Nigeria's hair and beauty industry moves billions of naira with zero digital trust infrastructure. Clients can't verify quality or lock prices before committing. Stylists lose income to no-shows and last-minute cancellations. Both sides take risks every single transaction.

## The Solution

GlamGo is a visual hair marketplace where clients discover hairstyles, find verified stylists nearby, and book with escrowed payment via Interswitch. The stylist's money is guaranteed. The client's price is locked. Trust is built into every transaction.

---

## Key Features

- **Premium Hair Gallery** — Beautiful, curated hairstyles with categories and filters
- **Stylist Profiles** — Portfolio, pricing, star ratings, service mode badges
- **Nearby Stylist Search** — Google Maps API for proximity-based discovery
- **Smart Booking** — Select available time slots directly from stylist's schedule
- **Secure Payment** — Interswitch-powered escrow holds funds until service is confirmed
- **Confirmation Code** — Mutual verification code when client and stylist meet
- **Service Verification** — Client confirms completion to release payment
- **Post-Service Rating** — Star ratings and reviews after each appointment
- **Client Hair Profile** — Hair type, length, and texture for stylist preparation
- **Service Mode Badges** — "Come to me," "I come to you," or "Both"
