# GlamGo Visual Quality Rubric

Use this rubric to self-evaluate every page and component before it ships.

## Score: 1-10

### 10 — Magazine Quality

- Looks like it belongs in a fashion magazine
- Animations are smooth and purposeful
- Typography hierarchy is perfect
- Images are the star — UI enhances them
- Mobile experience is flawless

### 8-9 — Premium

- Clearly a premium experience
- Good visual hierarchy and spacing
- Smooth interactions
- Minor polish opportunities

### 6-7 — Acceptable  

- Functional and clean
- Missing the "wow" factor
- Could pass as a standard web app
- Needs more visual personality

### 4-5 — Below Standard

- Looks like a generic template
- Poor spacing or typography choices
- No animations or transitions
- Images feel like an afterthought

### 1-3 — Unacceptable

- Looks broken or unfinished
- No consistent design language
- Would embarrass the team in a demo

---

## Quick Checklist

- [ ] Does this look like a fashion/beauty platform, not a SaaS tool?
- [ ] Are images given 70%+ of the visual space?
- [ ] Is there enough whitespace?
- [ ] Are fonts premium (not browser defaults)?
- [ ] Do interactive elements have hover/active states?
- [ ] Is the mobile layout polished, not just squeezed?
- [ ] Would Bukky (the hairstylist) be proud to show this to her clients?
