# Category Guide & Taxonomy
**Author:** Oyewole Oluwabukola Oghenerukevwe

## Overview
This document defines the client-facing category structure used for navigation and stylist organization within the GlamGo platform. It is designed to improve browse discoverability.

## Client-Facing Categories
1. **Braids**
2. **Kids Hairstyles**
3. **Locs (Dreads & Faux Locs)**
4. **Natural Hair Styling**
5. **Twists**
6. **Weaving (Cornrows & Stitch Styles)**
7. **Wigs & Installation**

## Asset Management Recommendation
Store all category and style images under the following organized path structure:
`public/images/hairstyles/<category>/`

Example:
`public/images/hairstyles/braids/knotless-1.jpg`
