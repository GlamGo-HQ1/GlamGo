# Component Tree

## Shared/Layout Components

- `Navbar` — Logo, navigation links, auth button
- `Footer` — Links, copyright
- `Container` — Max-width wrapper
- `Button` — Primary, secondary, outline variants
- `Input` — Text input with label and error state
- `Select` — Dropdown select
- `Modal` — Overlay modal
- `LoadingSkeleton` — Shimmer loading placeholder
- `EmptyState` — "No results" state with illustration
- `StarRating` — Displays/inputs star ratings
- `PriceTag` — Formatted price display (₦XX,XXX)
- `CategoryTag` — Rounded pill for categories
- `Avatar` — User/stylist profile photo
- `Toast` — Notification toasts

## Gallery Page Components

- `HeroSection` — "Find Your Next Look" header with search
- `CategoryFilter` — Horizontal scrollable category pills
- `HairstyleGrid` — Grid layout of hairstyle cards
- `HairstyleCard` — Individual style card (image, name, price, duration)

## Style Detail Page Components

- `StyleImageGallery` — Multiple images with carousel/grid
- `StyleInfo` — Name, category, description, price, duration
- `StylistList` — "Stylists Who Do This Style" section
- `StylistCard` — Mini stylist card (avatar, name, rating, price)

## Stylist Profile Page Components

- `StylistHeader` — Avatar, name, bio, rating, specialties
- `StylistPortfolio` — Grid of portfolio images
- `StylistReviews` — List of reviews
- `ReviewCard` — Individual review (client name, rating, comment)
- `StylistStyles` — Styles this stylist offers with prices

## Booking Flow Components

- `BookingSummary` — Style + stylist summary card
- `DatePicker` — Calendar date selection
- `TimeSlotPicker` — Available time slots
- `LocationInput` — Address input (text or map picker)
- `PriceSummary` — Breakdown of costs
- `BookingConfirmation` — Success page after booking

## Payment Components

- `PaymentSummary` — Order details before payment
- `PaymentButton` — "Pay with Interswitch" button
- `PaymentStatus` — Success/failure/pending display

## Dashboard Components

- `BookingList` — List of bookings (upcoming + past)
- `BookingCard` — Individual booking (style, stylist, date, status)
- `QuickActions` — Rebook, review, cancel buttons

## Auth Components

- `AuthForm` — Login/register form
- `RoleSelector` — Client vs Stylist role selection
